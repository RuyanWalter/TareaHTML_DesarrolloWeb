Mi sitio Web (Tarea No.1_Desarrollo Web)

En esta tarea vamos a utilizar las siguientes Etiquetas:

Etiqueta DIV

La etiqueta div es un elemento de bloque HTML que se utiliza para crear un contenedor de contenido en una página web.

La princiopal funcion de la etiqueta div es organizar y agrupar elementos dentro de una página web para darle una estructura clara y estética.

Etiqueta SECTION

La etiqueta Section se utiliza para identificar una seccion distinta y coherente en un documento HTML.

Etiqueta H1

H1 es una etiqueta de encabezado en HTML5 que se utiliza para definir el título principal de una página web

Etiqueta H2

H2 es una etiqueta de encabezado en HTML5 que se utiliza para definir subtitulos secundarios debajo del título principal
